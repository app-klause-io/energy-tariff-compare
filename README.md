# Energy Tariff Compare

Compare energy tariffs based on your actual usage profile. Built for EV owners, heat pump homes, and anyone tired of generic comparison sites.

## Development

```bash
pnpm install
pnpm dev
```

## Testing

```bash
pnpm lint        # Prettier + ESLint
pnpm check       # TypeScript + svelte-check
pnpm test:unit   # Vitest
pnpm build       # Production build
```
